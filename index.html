개인정보처리방침

루티니 (Routiny) - 습관 트래커
📌 요약
루티니는 사용자의 개인정보를 수집하지 않으며, 모든 습관 데이터는 사용자의 기기에만 저장됩니다.

1. 개인정보의 수집 및 이용

1.1 수집하는 정보

루티니는 다음과 같은 정보를 처리합니다:

수집 항목 수집 방법 보관 장소
습관 이름, 아이콘, 색상 사용자 직접 입력 사용자 기기 내부
습관 완료 기록 (날짜) 사용자 체크인 시 사용자 기기 내부
앱 설정 정보 자동 생성 사용자 기기 내부

1.2 개인정보를 수집하지 않습니다

루티니는 다음 정보를 수집하지 않습니다:

  • 이름, 이메일 등 개인 식별 정보
  • 전화번호
  • 위치 정보
  • 연락처
  • 사진 또는 미디어 파일
  • 기기 식별 정보 (광고 ID 제외)

1.3 데이터 저장 방식

💾 로컬 저장소만 사용
모든 습관 데이터는 사용자의 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

2. 제3자 서비스

2.1 Google AdMob

루티니는 광고 표시를 위해 Google AdMob을 사용합니다.

AdMob이 수집할 수 있는 정보:

  • 광고 ID (익명)
  • 기기 정보 (모델명, OS 버전)
  • 앱 사용 통계
  • 광고 상호작용 정보

Google의 개인정보처리방침:
https://policies.google.com/privacy

맞춤 광고 거부 방법:

  • Android: 설정 → Google → 광고 → 광고 맞춤설정 선택 해제
  • iOS: 설정 → 개인정보 보호 → 추적 → 앱이 추적을 요청하도록 허용 끄기

3. 개인정보의 보유 및 이용 기간

사용자가 앱을 사용하는 동안 데이터가 기기에 보관되며, 다음의 경우 즉시 삭제됩니다:

  • 앱 삭제 시
  • 앱 내 데이터 삭제 기능 사용 시
  • 기기 초기화 시
⚠️ 중요
현재 버전은 클라우드 백업 기능이 없으므로, 앱을 삭제하면 모든 데이터가 영구적으로 삭제됩니다.

4. 개인정보의 제3자 제공

루티니는 사용자의 개인정보를 제3자에게 제공하지 않습니다.

단, 다음의 경우 예외적으로 정보가 공유될 수 있습니다:

  • 법률에 의해 요구되는 경우
  • 사용자의 생명, 신체, 재산 보호를 위해 필요한 경우

5. 개인정보의 파기

루티니는 외부 서버에 데이터를 저장하지 않으므로, 앱 삭제 시 모든 데이터가 자동으로 파기됩니다.

6. 아동의 개인정보 보호

루티니는 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 만약 만 14세 미만 아동의 정보가 수집되었음을 발견한 경우, 즉시 해당 정보를 삭제합니다.

7. 이용자의 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 열람 권리: 앱 내에서 모든 습관 데이터 확인 가능
  • 수정 권리: 앱 내에서 습관 정보 수정 가능
  • 삭제 권리: 앱 내에서 습관 삭제 또는 앱 삭제로 모든 데이터 삭제

8. 개인정보 보호책임자
